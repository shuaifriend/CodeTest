# EmailJS Demo

使用JavaScript直接从前端发送电子邮件💥💥

体验Demo：[https://dunizb.github.io/CodeTest/project/emailjs/](https://dunizb.github.io/CodeTest/project/emailjs/)

## 使用

运行前请按照配套文章获取apikey。

运行：`npm run dev`

build：`npm run build`
